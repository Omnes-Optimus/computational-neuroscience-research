# Computational Neuroscience Research
### Transient Explosive Synchronization in Mesoscale Mouse Brain Networks

> Public overview of my research contributions.  
> The implementation code is maintained in a private research repository and is therefore not reproduced here.

---

## Overview

This project investigates **transient Explosive Synchronization (tES)** in biologically inspired brain networks.

The objective is to understand how synchronization emerges through multiple coexisting pathways in networks of coupled oscillators under adaptive coupling and resource constraints.

The simulations are performed on a mesoscale mouse brain connectome derived from the Allen Mouse Brain Atlas.

- Managing experimental outputs for further analysis
