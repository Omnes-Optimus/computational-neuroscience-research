# Computational Neuroscience Research
### Transient Explosive Synchronization in Mesoscale Mouse Brain Networks

> Ongoing undergraduate research project in computational neuroscience.

---

## Overview

This repository serves as a public overview of my ongoing research work on **transient Explosive Synchronization (tES)** in biologically inspired brain networks.

The implementation is maintained in a private collaborative repository. This repository documents the project, my contributions, and the scientific context without exposing unpublished research code.

---

## Research Objective

The project investigates how synchronization emerges in complex brain networks through multiple coexisting pathways.

The work combines:

- Kuramoto-inspired oscillator dynamics
- Adaptive (STDP-inspired) coupling
- Resource dynamics
- Mesoscale mouse brain connectomes
- Numerical simulation and data analysis

---

## Current Responsibilities

My work currently includes:

- Running large-scale parameter sweeps
- Generating simulation datasets
- Studying synchronization transitions
- Computing synchronization metrics
- Analyzing first-passage times
- Developing analysis notebooks
- Interpreting simulation results
